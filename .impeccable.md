## Design Context

### Users
Recruiters, design leads, collaborators, and peers who need to quickly understand Left2y as a UI/UX designer, visual creator, and AI-app builder. They usually scan the site on desktop first, then revisit specific work on mobile.

### Brand Personality
Precise, kinetic, curious. The interface should feel like a living studio index: enough structure to read quickly, enough texture to signal that the work is visual, experimental, and hands-on.

### Aesthetic Direction
An editorial field archive for a Guangdong-based digital creator, built around the original warm yellow paper background and orange signature color. Light mode is the default because the portfolio is mostly read in daytime review contexts; dark mode remains available for evening browsing. The direction should avoid generic glass cards, neon tech dashboards, and template portfolio grids.

### Design Principles
- Lead with real work imagery instead of abstract decoration.
- Make scanning fast: visible categories, counts, roles, and direct actions.
- Keep the system tactile with borders, stamps, and compact metadata, but never let the chrome overpower the work.
- Prefer progressive disclosure: overview first, case details and media controls only when selected.
- Make contact and resume actions immediate, copyable, and accessible.
